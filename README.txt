1. Project Description

This JavaScript/React web app displays order data from the QuickBooks accounting software (using their API), and allows users to input the weights/dimensions for each order, allowing for a smoother production/shipping workflow.


----------
2. Requirements

React.js


----------
3. Installation:

-Install React 18.2 or higher
-Connect to QuickBooks API (ensure folder permissions are correct)
-Create a local .env file with the credentials listed in lines 10-16
-if uploading to git: create a .gitignore file with the text '.env' inside
